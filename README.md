# FizzBuzz - Proyecto TDD

Implementación del juego FizzBuzz con **Test-Driven Development** (TDD) con pruebas descriptivas

## Descripción

El juego FizzBuzz retorna:
- `"Fizz"` si el número es divisible por 3
- `"Buzz"` si es divisible por 5
- `"FizzBuzz"` si es divisible por 15
- El número como string en otros casos


## Ejecución

```bash
# Compilar y pruebas
mvn clean test

# Solo compilar
mvn compile

# Solo pruebas
mvn test
```

## Requisitos

- Java 17+
- Maven 3.6+
